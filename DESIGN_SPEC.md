# 데이터 과학 시리즈 — 공통 디자인 스펙

> 4개 단원 페이지(unit1~4.html)가 한 시리즈로 보이게 하는 고정 장치.
> 단일 HTML 파일 · 오프라인 더블클릭 실행 · CDN/웹폰트/이미지 0 · 순수 HTML/CSS/JS.
> 새 단원을 만들 때 이 문서와 unit1.html의 컴포넌트 문법을 그대로 계승할 것.

## 팔레트 (검증 완료 — 임의 변경 금지)

- **차트 카테고리 5색 (고정 순서, 순환 금지)**: `#2563EB` `#D97706` `#0891B2` `#7C3AED` `#DB2777`
  - dataviz 검증: 밝기 밴드·채도 하한·CVD 분리(ΔE 51+)·표면 대비 **전부 PASS** (surface `#FAFBFD`, 2026-07-10)
- **단원 액센트 = 위 5색에서 배정** (시리즈 장치: "단원 색 = 차트 팔레트의 구성원"):
  - 1단원 이해 = `#0891B2` 시안 · 2단원 준비·분석 = `#D97706` 앰버 · 3단원 모델링 = `#7C3AED` 바이올렛 · 4단원 프로젝트 = `#DB2777` 로즈
  - CSS에서는 `--ua`(unit accent) / `--ua-soft` / `--ua-dark` 3종 토큰으로만 참조 — 단원 바꿀 때 이 3줄만 교체
- **중립**: bg `#FAFBFD` · 카드 `#fff` · 잉크 `#17263B` · 보조 `#5D6B7E` · 라인 `#E4E9F0`
- **시맨틱**: 정답/성공 `#0E9F6E` · 오답/경고 `#E02424` (색+아이콘 병행, 색 단독 금지)

## 차트 규칙 (dataviz 스킬 준수)

- 한 측정값의 막대 비교 = **단일 색조**(단원 액센트) + 강조 1개만 진하게, 나머지는 연하게. 막대마다 다른 색 금지.
- 다계열(산점도·군집 등)만 카테고리 5색을 **고정 순서로** 사용. 값 라벨·범례 텍스트는 잉크색(계열색 금지).
- 막대 데이터 끝 4px 라운드, 이중 축 금지, 무지개 금지.

## 컴포넌트 문법 (unit1.html이 원본)

- `.card` 개념 카드 + `.chip`(1-1, 1-2…) / `.tip` 꿀팁 / `.warn` 오개념 주의보
- `.demo` 인터랙티브 데모 + `.demo-title`에 LIVE 배지 — **단원당 데모 2~3개** (원페이지가 아닌 이유가 데모이므로 아끼지 말 것)
- 퀴즈: QUIZ 배열 + renderQuiz() 엔진(info_science_master 계승) — 아코디언 1개, 단원당 5문항
- 헤더: 도트 산점 모티프(CSS radial-gradient) + "데이터 과학 · N단원 / 4" 배지 + 미니 목차 칩
- 진도/기록: localStorage 미사용(세션만) — 교실 공용 PC 전제

## 톤

- 중~고등학생, 존댓말 아닌 "~다/~보자" 서술, 비유 우선(매점·급식·도서관 등 학교 생활 소재), 이모지는 제목·포인트에만
- 성취기준 코드([12데과…])는 페이지에 명시하지 않는다 — 내용으로만 커버
- 15px 미만 본문 금지, prefers-reduced-motion 전면 대응, 버튼 focus-visible 필수
